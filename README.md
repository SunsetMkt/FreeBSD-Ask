## 概述

FreeBSD 从入门到跑路诞生于 2021 年 12 月 19 日。~~为了拯救即将倒闭的 FreeBSD.....我们决定写一本书~~

本书定位：编写一本 FreeBSD 版本的《鸟哥的 Linux 私房菜：基础学习篇》&《鸟哥的 Linux 私房菜：服务器架设篇》。

**编辑指南概要**：我们欢迎所有支持 FreeBSD 的人进行编写，并会将其列入贡献者名单。
